Java — Core Concepts

Short notes to get started with Java. Practice each example and build small apps.

1. OOP Concepts

Class, Object, Inheritance, Polymorphism, Abstraction, Encapsulation.

class Student {
  String name;
  void show() { System.out.println(name); }
}

2. Collections Framework

  • List — ArrayList, LinkedList
  • Set — HashSet, TreeSet
  • Map — HashMap, TreeMap
  • Queue — PriorityQueue
ArrayList list = new ArrayList<>();
list.add("Java");
System.out.println(list);

3. Exception Handling

Try, Catch, Finally, Throw, Throws — prevents program from crashing.

try {
   int x = 10 / 0;
} catch (Exception e) {
   System.out.println("Error: " + e);
}

4. Multithreading

Thread class, Runnable interface, synchronization.

class MyThread extends Thread {
  public void run() {
    System.out.println("Thread running...");
  }
}

5. File Handling

Read / write files using File, FileWriter, Scanner.

FileWriter fw = new FileWriter("abc.txt");
fw.write("Hello Java");
fw.close();

6. JDBC (Database Connection)

Used to connect Java program with database — MySQL, Oracle, etc.

Class.forName("com.mysql.jdbc.Driver");
Connection con = DriverManager.getConnection(url, user, pass);

7. Java Important Features

  • Platform Independent
  • Simple & Secure
  • High Performance
  • Robust & Portable
  • Object Oriented
