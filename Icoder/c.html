C — Core Notes

Pointers, memory management, file IO and system-level programming basics.

1. Pointers & Memory

Pointer arithmetic, malloc/free, stack vs heap, pointer to pointer, NULL pointer and void pointer usage.

2. Arrays & Strings

1D & 2D arrays, memory layout, character arrays vs string functions, string.h library functions (strcmp, strcpy, strlen).

3. Functions & Recursion

Pass-by-value vs pass-by-reference, inline recursion, tail recursion, static variables inside functions.

4. Structures & Unions

Defining structure, typedef, nested structures, difference between structure & union memory size, padding & alignment.

5. File Handling

fopen, fclose, fread, fwrite, fgets, fputs, file pointer, reading/writing binary & text file, error handling using errno.

6. Dynamic Memory Allocation

malloc, calloc, realloc, free, memory leak, dangling pointer, sizeof operator in memory allocation.

7. Preprocessor & Macros

#define, #include, conditional compilation (#ifdef/#ifndef), multi-line macros, predefined macros (__DATE__, __TIME__).

8. Command Line Arguments

argc, argv usage, passing file paths and values through terminal, converting string to int using atoi().

9. Makefile & Compilation Process (Bonus)

gcc compile pipeline: Preprocessing → Compilation → Assembling → Linking. Makefile uses rules + targets to automate build.
